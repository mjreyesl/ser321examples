You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=#&num2=# -- multiply 2 values
  • host:PORT/github?query=users/amehlhase316/repos -- contain public repos of user
  • /primeFactor?number=#&min_factor=# -- calculates prime factors of a number with a minimum factor
  • /fibonacci?n=#&start=# -- request will return the Fibonacci sequence up to the n-th number, starting from 0 or 1 only.
